# Instalar los paquetes para el cuaderno
Para instalar los paquetes para trabajar en el repositorio es necesario contar con los paquetes necsarios y las versiones correctas, para ello basta con ejectuar el siguiente comando dentro de la carpeta contenedora:
```
conda env create -f tf.yml
```
Para verificar que los paqutes se hayan instalado correctamente basta con listar los ambientes instalados con el sigueinte comando:
```
conda list
```
